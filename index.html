      Export

      Ouvrez ce fichier dans Google Earth (Fichier > Ouvrir), ou déposez-le dans votre Google Drive pour le retrouver depuis Google Earth Web. Vous pourrez ensuite le convertir en DXF/DWG via QGIS ou AutoCAD Map 3D, comme on l'a vu précédemment.

      Si le téléchargement ne se déclenche pas dans l'application (selon l'appareil), le fichier s'ouvrira automatiquement dans un nouvel onglet : utilisez alors le menu « Partager » ou « Enregistrer sous » de votre téléphone pour le sauvegarder dans vos fichiers ou l'envoyer par email/WhatsApp.

      Les données restent sauvegardées dans cette conversation Claude tant que vous ne réinitialisez pas. Si vous ouvrez ce fichier seul dans un navigateur (hors Claude), exportez le KML avant de fermer la page pour ne rien perdre.
